Languages I Code In

  • HTML

    HTML is a markup language used for creating web pages and web applications. It is the standard markup language used to create web pages.

  • CSS

    CSS is a style sheet language used for describing the look and formatting of a document written in HTML or XML. It is used to control the layout and appearance of web pages.

  • JavaScript

    JavaScript is a high-level, interpreted programming language that is primarily used to enhance web pages to provide for a more user-friendly experience.

  • Java

    Java is a general-purpose programming language that is class-based, object-oriented, and concurrent. It is one of the most popular programming languages in use today.

  • SQL

    SQL is a standard language for managing and manipulating databases. It is used to create, modify, and query databases.
